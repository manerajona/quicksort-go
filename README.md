# quicksort-go

Quick sort is basically based on divide and conquer method. It divides the array into two parts and recursively solves them.
    
## Time Complexity

- Best case:  O(n log(n))

- Average case:  O(n log(n))

- Worst case:  O(n*2)

## Space Complexity

- Worst case:  O(log(n))


## Build & Run

For building:
```sh
$ go build main.go
```
For running:
```sh
$ go run main.go
```
